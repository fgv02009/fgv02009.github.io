GIT Cheat Sheet

Create Repositories Make Changes Branches & Merging Relocate & Remove Files Store Work in Progress History Create & Clone Repo Undoing Changes Merge Conflict Shortcuts & Emoticons

Create Repositories

    git init [project-name]

  • creates new local repo with specific

    • git clone [url]

  • to download project from Github

Make Changes

    git status

  • list new and modified files

    • git diff

  • shows file diff. that have not been saved

    • git add [file]

  • preparing to commit file

    • git add '*.file type'

  • adds a specific file type

    • git diff --staged

  • shows file diff between staging and last file

    • git commit --a

  • commit all local changes in tracked files

    • git commit --m [message]

  • records file permanently

    • git push [origin][master]

  • pushing changes to the master repo

Branches and Merging

    git branch

  • lists local branches in current repo

    • git branch [name]

  • creates new branch

    • git checkout [branch-name]

  • switches to specified branch and updates directory

    • git merge [branch]

  • combines specified branch history into current branch

    • git branch --d [branch name]

  • deletes specified branch

Relocate and Remove Files

    git rm [file]

  • deletes file from working directory and stages deletion

    • git rm --cached [file]

  • removes file from version control but preserves file locally

    • git mv [file original] [file renamed]

  • changes file name and prepares it for commit

Store Work in Progress

    git stash

  • temporarily stores modified tracked files

    • git stash pop

  • restores most recently stashed files

    • git stash list

  • list all stashed changesets

    • git stash drop

  • discards the most recently stashed changeset

History

    git log

  • shows history of commits for current branch

    • git log --p [file]

  • shows changes over time for a specific file

    • git log --follow [file]

  • lists version history for a file, including renames

    • git diff [first branch]...[second-branch]

  • shows content differences between two branches

    • git show [commit]

  • outputs metadata and content changes of specified commit and can use SHA

    • git blame [file]

  • checks who changed what and when in file

Create and Clone Repo

    git clone [url]

  • create a copy of an existing Git repo into a newly created directory

    • git remote --v

  • lists all currently configured remotes

    • git remote show [remote]

  • info about a remote

    • git remote add [remote][url]

  • add new remote repository

    • git fetch [remote]

  • downloads all changes from remote but not integrated in head

    • git pull [remote][branch]

  • downloads changes and directly merges into HEAD

    • git push [remote][branch]

  • publish local changes on a remote

    • git push [remote] :[branch]

  • delete a branch on remote

Undoing Changes

    git reset [file]

  • unstages file and preserves contents

    • git reset --hard HEAD

  • discard all local changes in working directory

    • git checkout HEAD [file]

  • discard local changes in a specific file

Merge conflict

    git mergetool

  • solve conflicts

    • git add [resolved file]

    git rm [resolved file]

  • manually solve conflicts and make files as resolved

Keyboard Shortcuts

  • t bring up a file explorer
  • w bring up the branch selector
  • s select the Command Bar
  • l edit labels on
  • n go to notifications
  • d go to Dashboard
  • ? see all of the shortcuts for the current page

Emoticons

  • :smiley:
  • :+1:
  • :-1:
  • :confused:
  • :question:
