Intro to SQL with Cheat Sheet

by: Flori Garcia-Vicente and Andrew Irwin

SQL (pronounced sequel) stands for structured query language. It is a special purpose programming language designed to manage and communicate with data held in a relational database system (RDBMS).

SELECT

        SELECT column_name,column_name
        FROM table_name;
      

        SELECT state_name, population
        FROM states;

The code above will return the states table with two columns: state_name and population.

WHERE

        SELECT column_name, column_name
        FROM table_name
        WHERE column_name operator value;
      

        SELECT state_name, population_density, population
        FROM states
        WHERE population > 2000000;

This code will return the states table with three columns: state_name, population_density, and population for only the states with a population greater than 2,000,000.

ORDER BY

        SELECT column_name, column_name
        FROM table_name
        ORDER BY column_name ASC|DESC
      

        SELECT state_name, population_density, population
        FROM states
        WHERE population > 2000000
        ORDER BY population ASC;

This code will return the states table with three columns: state_name, population_density, and population for only the states with a population greater than 2,000,000 and orders the table in ascending order by population size.

Wildcard

  • % - A substitute for zero or more characters
  • _ - A substitute for a single character
  • [charlist] - Sets and ranges characters to match
  • [^charlist] or [!charlist] - Matches only a character not specified within the brackets
        SELECT region_name
        FROM regions
        WHERE region_name LIKE '%South%';

This code will return the regions table with the column region_name for regions that include the word South.

AND/OR

        SELECT state_name, region_id
        WHERE population > 2000000
        AND region_id > 5;

This code will return the states table with two columns state_name and region_id, where population is greater than 2000000 and region_id is greater than 5.

        SELECT state_name, region_id
        WHERE population > 2000000
        OR region_id > 5;

This code will return the states table with two columns state_name and region_id, where population is greater than 2000000 OR region_id is greater than 5.

Inner Join

        SELECT regions.region_name, states.state_name
        FROM regions
        INNER JOIN states
        ON regions.id=states.region_id
        ORDER BY region_id;

This code returns a table with two columns, region_name and state_name, in ascending order by region_id.
